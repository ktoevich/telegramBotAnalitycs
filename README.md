# Analytics Lesson Summarizer Bot 🎓

Этот Telegram-бот предназначен для мониторинга обсуждений во время уроков. Он сохраняет сообщения учащихся и с помощью искусственного интеллекта (Google Gemini) формирует краткий, емкий и читаемый конспект основных идей и выводов.

## Особенности
- **Синхронизация по датам**: Разделяет обсуждения по дням. Вы можете получить конспект как за текущий день, так и за любой прошедший урок командой `/summary YYYY-MM-DD`.
- **Асинхронность**: Бот не "зависает" при ожидании генерации конспекта.
- **Интеллектуальный анализ**: Автоматически очищает рассуждения от "воды", убирает конкретные имена участников (если они не важны) и оставляет сухую и самую ценную информацию с помощью модели `gemini-flash-latest`.

## Запуск проекта локально

### Требования
- Текущая версия Python: **3.11+**
- Telegram Bot Token
- Google Gemini API Key

### Установка

1. Склонируйте репозиторий и перейдите в папку.
2. Создайте виртуальное окружение:
```bash
python -m venv venv
venv\Scripts\activate  # Для Windows
# Вариант для Linux/Mac: source venv/bin/activate
```
3. Установите зависимости:
```bash
pip install -r requirements.txt
```
4. Скопируйте файл `.env.example` в `.env` и вставьте ваши ключи API:
```env
TELEGRAM_BOT_TOKEN=ваш_токен
GEMINI_API_KEY=ваш_ключ
```
5. Запустите бота:
```bash
python main.py
```

## Использование
- `/start` - Запустить бота и получить информацию.
- `/help` - Показать доступные команды.
- `/summary` - Получить конспект переписки за сегодняшний день.
- `/summary 2024-05-12` - Получить конспект за конкретную дату.
